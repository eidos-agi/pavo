# Code of Conduct

Pavo follows the Eidos AGI expectation for direct, respectful, evidence-backed
collaboration.

Contributors should:

- Treat people with respect.
- Keep technical disagreements specific and evidence-based.
- Avoid publishing private recordings, sensitive metadata, credentials, or
  personal data.
- Report security or privacy concerns through the security process instead of
  public issues.

Maintainers may remove contributions or access that undermine these standards.
